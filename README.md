# Freight Truck Perception Module

## Overview
Aim of this project is to implement a perception module for freight truck driver assistance using computer vision. I will compare a few models and datasets to decide the best approch.

## Models Used
- YOLO (Object Detection)
- DeepLabV3+ (Semantic Segmentation)
